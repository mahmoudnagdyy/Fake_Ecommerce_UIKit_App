# 🛍️ Fake E-Commerce UIKit App

A simple iOS application built using **UIKit** that simulates an e-commerce experience with product listing, searching, and editing capabilities.

🔗 **Repository:**
https://github.com/mahmoudnagdyy/Fake_Ecommerce_UIKit_App

---

## 📱 Features

* 🧾 Display products using **UITableView**
* 📦 Display Product Detail Using **Navigation Controller**
* 🔍 Search functionality using **UITextField**
* ✏️ Edit & delete items with **swipe actions**
* ♻️ Dynamic filtering of products
* 🧠 Clean architecture using **MVVM**
* 🔌 Dependency Injection for better scalability

---

## 🏗️ Architecture

This project follows the **MVVM (Model-View-ViewModel)** pattern:

* **Model**

  * Represents product data (fake/local data)

* **View**

  * UIKit components (UITableView, UITextField, UIButton)

* **ViewModel**

  * Handles business logic
  * Connects View with Model
